# Blog Project — Sistemas y Tecnologías Web

Aplicación web tipo blog construida con HTML, CSS y JavaScript vanilla.
Usa la API de DummyJSON para listar, buscar y gestionar publicaciones.

## Integrantes
- Fatima Navarro 
- Juan Jose Rivas

## Estructura
\`\`\`
css/          → estilos separados por responsabilidad
js/api/       → llamadas fetch a DummyJSON
js/pages/     → lógica de cada vista
js/utils/     → helpers de DOM y validación
\`\`\`

## Páginas
- `index.html` — listado de posts con búsqueda, filtros y paginación
- `detail.html` — detalle de una publicación
- `create.html` — formulario para crear post (POST)
- `profile.html` — perfil de usuario/autor

## API
[DummyJSON](https://dummyjson.com/docs/posts)
